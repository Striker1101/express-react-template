Project Overview

This project is a full-stack application consisting of a React frontend and an Express backend, dockerized for development and production use.

Setup and Installation

Frontend Setup

  1. Navigate to the Frontend Directory: cd frontend
  2. Install Dependencies: npm install
  3. Run the Development Server: npm start
  4. Build for Production: npm run build

Backend Setup

  1. Navigate to the Backend Directory: cd backend
  2. Install Dependencies: npm install
  3. Run the Development Server: npm run dev

Frontend Development

The frontend is built using React, Tailwind CSS, and Axios. Key files include src/App.js and tailwind.config.js.

Dark mode toggle is implemented via a state variable in App.js.

Backend Development

The backend is built using Express, CORS, Dotenv, and Cross-Env. Key files include app.js and routes/api.js.

Environment variables are stored in a .env file.

Dockerization

The application is dockerized using separate Dockerfiles for the frontend and backend, and a Docker Compose file for multi-container deployment.

Deployment

To deploy the application, use Docker Compose with the command: docker-compose up --build.

Useful Docker commands include docker-compose build and docker-compose down.

Configuration and Environment Variables

Frontend environment variables are set using REACT_APP_BACK_END_URL.

Backend environment variables are managed in a .env file.

Troubleshooting

Common issues include port conflicts, dependency errors, and Docker issues. Use commands like docker logs <container_id> for debugging.
